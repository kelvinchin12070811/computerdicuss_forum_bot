## 安装方法

--系统前置程序要求--
- [ ] [Node.js](https://nodejs.org/zh-cn/) [最低版本：16.9.1] 
- [ ] [Git](https://gitforwindows.org/)[最低版本：2.33.0]


```
git clone https://github.com/kelvinchin12070811/computerdicuss_forum_bot.git
npm -g install yarn
yarn install
yarn build
node ./build/index.js
```
